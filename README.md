# Student-Admin
### 项目说明
    一、项目简介
        名称：教务管理系统
        技术栈：
            -springboot 2.1.15
            -mysql 8.0.20
            -mybatis
            -redis（完成了配置，仅对用户登录进行了缓存处理）
            -thymeleaf
            -shiro
        功能：
            1.管理员：对学生、教师个人信息进行管理，对教师添加的课程统一发布
            2.教师端：完成课程的相关操作，针对课程发布考试，并录入成绩
            3.学生端：完成选课，考试信息查询，成绩查询等功能
        
    二、项目启动
        环境配置：
            -maven3.6.3
                brew install maven
            -java1.8
                brew cask install java
            -docker（可选）
                官网下载
        主程序启动
